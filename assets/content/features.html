Ready to build for serious fun games?!

Heaps.io delivers fast iterations, real development power and multi-platform compilation without overhead. The toolkit is versatile, open-source and completely free. It only requires your creativity to build that fun game you have in mind.

A friendly API with several purposes

The Heaps.io API is comprised of several top level packages, namely:
  • h2d used for 2D display (for 2D games and user interfaces)
  • h3d used for rendering 3D models
  • hxd contains cross platform classes such as Bitmaps, and a complete resource loading and management framework
  • hxsl is the Heaps Shader Language implementation

Fast builds

Big projects with ~100 loc compile in seconds with the compilation cache server. Test in browser, build the app later.
