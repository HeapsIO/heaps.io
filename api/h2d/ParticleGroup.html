Should group rebuild on parameters change. Note that some parameters take immediate effect on the existing particles, and some would force rebuild reagrdless. Parameters that take immediate effect: speedIncr, gravity, gravityAngle, fadeIn, fadeOut, fadePower, rotAuto, rotInit, incrX, incrY, emitLoop, blendMode Parameters that will always force rebuild: enable, sortMode, isRelative, texture, frameCount, frameDivisionX, frameDivisionY, nparts Parameters that newer cause rebuild: blendMode, colorGradient, animationRepeat
